# Mohammednur Seid | Portfolio

A modern, high-performance personal portfolio website built with a focus on clean UI/UX and interactive customization. This project showcases the skills, services, and selected works of Mohammednur Seid, an Information Technology student and Full Stack Developer at Bahir Dar University.

---

## 🚀 Features

* **Dynamic Typing Effect**: An engaging hero section that cycles through professional titles.
* **Customization Sidebar**: A unique user interface that allows visitors to personalize their experience in real-time.
    * **Theme Switching**: Support for Light and Dark modes.
    * **Environment Presets**: Options to toggle between different visual backgrounds (Constellation, Neural, Minimalist).
* **Interactive Design**: 
    * Smooth reveal-on-scroll animations using the Intersection Observer API.
    * Custom scroll indicator.
    * Glassmorphism effects in Dark Mode.
* **Responsive Layout**: Fully optimized for desktop, tablet, and mobile devices.
* **Service & Project Showcases**: Categorized grids highlighting technical expertise and key software builds.

---

## 🛠️ Tech Stack

* **HTML5**: Semantic structure for accessibility and SEO.
* **CSS3**: Custom properties (CSS variables), Flexbox/Grid layouts, and advanced animations.
* **JavaScript (ES6+)**: Vanilla JS for DOM manipulation, typing logic, and theme persistence.
* **Icons**: SVG-based iconography for crisp visuals on all resolutions.
* **Fonts**: *Plus Jakarta Sans* via Google Fonts.

## 🎨 Themes & Customization

The portfolio features a robust theme engine. In **Dark Mode**, the site transforms into a "Cyber" aesthetic, utilizing a deep blue/black palette with neon cyan accents (`#00fbff`) and a localized background image for a futuristic feel. 

Users can access the customization menu via the **settings gear** in the navigation bar to reset styles or experiment with different "Environment" grids.
