# Mohammednur Seid | Professional Portfolio

Welcome to the source code for my professional portfolio website! This project is a modern, responsive personal portfolio designed to showcase my skills, projects, and services as a Full Stack Developer and UI/UX Designer.

## 🔗 Live Demo
Check out the live website here: [MyPortfolio](https://mohammednurseid46.github.io/MyPortfolio/)

## ✨ Features
- **Modern & Premium Design:** Sleek UI featuring custom animations and interactive elements.
- **Theme Customization:** 
  - Light & Dark mode support.
  - Interactive background environments (Constellation Grid, Neural Network, Minimalist).
- **Responsive Layout:** Optimized for seamless viewing on all devices (mobile, tablet, and desktop).
- **Interactive Sidebar:** Easy access to personalization and styling settings.
- **Project Showcase:** Highlights selected works and accomplishments.

## 🛠️ Built With
- **HTML5:** Semantic and accessible structuring.
- **CSS3:** Custom styling, CSS Variables for theming, Flexbox/Grid layouts, and smooth animations.
- **JavaScript (Vanilla):** Dynamic interactions, custom cursor features, theme toggling, and interactive background environments.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/mohammednurseid46/MyPortfolio.git
   ```
2. **Open the project**
   Navigate to the project directory and open `index.html` in your favorite web browser. No complex build tools required!
